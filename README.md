# Gun Data Violence Visualization

A cool data visualizations with Deck.gl and Google Maps JS

## Running locally

```
git clone <this-repo>
npm install
npm start
```

NOTE: Don't forget to update the Google Maps API key in the `public/index.html` file.

The gundata.json file contains over 140,000 U.S. gun-related incidents from 2013-2018 from http://www.gunviolencearchive.org/
